### 🎯 **核心要点**

#### **引用强度排序**
```
强引用 > 软引用 > 弱引用 > 虚引用
```

#### **GC回收时机**
- **强引用**：永不回收（除非无可达路径）
- **软引用**：内存不足时回收
- **弱引用**：下次GC时回收
- **虚引用**：对象回收后通知

#### **典型应用场景**
- **强引用**：日常对象引用
- **软引用**：内存敏感缓存
- **弱引用**：规范映射、监听器
- **虚引用**：资源清理、内存监控
